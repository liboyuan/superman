-g全局 --svae更新package.json中dependencies -dev更新package.json中devDependencies
npm install -g cnpm --registry=https://registry.npm.taobao.org 淘宝镜像cnpm

npm install  在package.json文件中的dependencies中的插件全部下载

npm install <Module Name>  安装插件  

npm install <Module Name>@版本号 安装插件指定版本

npm uninstall <Module Name> 卸载插件

npm init 创建package.json文件

npm list <Module Name> 查看模块版本号

npm update <Module Name> 更新模块
